# Pixel Office Layouts

Community gallery of layouts for [Pixel Office](https://github.com/fedevgonzalez/pixel-office).

## How to use a layout

1. Browse the `layouts/` folder and pick one you like
2. In Pixel Office, open **Settings** > **Import Layout**
3. Select the downloaded `.json` file

## How to submit a layout

1. Fork this repo
2. Export your layout from Pixel Office (**Settings** > **Export Layout**)
3. Add your file to `layouts/` with a descriptive name (e.g. `cozy-startup-office.json`)
4. Add a screenshot to `screenshots/` with the same name (e.g. `cozy-startup-office.png`)
5. Update the gallery table below
6. Open a Pull Request

## Naming convention

```
layouts/{name}.json
screenshots/{name}.png
```

Use lowercase kebab-case. Keep names short and descriptive.
